# pgp-suffix-finder

Brute force PGP key ID suffixes of arbitrary length.

## Warning!

This tool can brute force any (short) PGP key suffix, use it for good; e.g.
creating vanity key IDs. Don't use it to generate collisions and annoy other
PGP users. Thanks
